A little history

Tim Berners Lee is credited with creating the world wide web and HTML.
The World Wide Web (W3) initiative links related information throughout the globe. HTML provide one simple format for providing linked information, and all W3 compatible programs are required to be capable of handling HTML. W3 uses an internet protocal(Hypertext Transfer Protocal, HTTP), which allow transfer representations to be negotiated between client and server, the result being returned in an extended MIMe message. HTML is therefore just one, but an important one, of the representations used with W3.
